go-helloworld-api
================

A repository structure for develping a go API.

### Getting started

* Run `make debug`

### Dependencies

* No further dependencies other than those defined in `go.mod`

### Configuration

| Environment variable         | Default   | Description                  |
| ---------------------------- | --------- | -----------------------------|
| BIND_ADDR                    | :25700    | The host and port to bind to |
